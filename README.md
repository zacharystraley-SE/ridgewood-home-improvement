# Ridgewood Home Improvement

Professional home improvement website for kitchen remodeling, bathroom remodeling, interior design, space planning, construction & project management, landscape design, and custom cabinetry services.

## Live Site

Visit the live website at: `https://yourusername.github.io/ridgewood-home-improvement`

(Replace `yourusername` with your actual GitHub username)

## Files

- **index.html** - Complete website (HTML, CSS, JavaScript all in one file)
- **README.md** - This file
- **.gitignore** - Tells Git which files to ignore

## Editing the Website

Edit `index.html` directly to make changes:

- **Business info**: Search for phone number, email, address, hours
- **Colors**: Look for `--navy-dark`, `--accent-blue`, `--accent-green` in the `<style>` section
- **Text content**: Edit section headings, descriptions, testimonials
- **Images**: Replace Unsplash image URLs in the hero carousel with your own
- **Services**: Update the 6 service cards with your offerings

## Deploying Updates

1. Edit `index.html` locally
2. Test in your browser
3. Go to your GitHub repository
4. Click the file name → click the pencil icon to edit
5. Paste your updated code
6. Click "Commit changes"
7. Your live site updates within seconds
